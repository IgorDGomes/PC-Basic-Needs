### Usage

1. Clone the repository:

```bash
    git clone https://github.com/IgorDGomes/PC-Basic-Needs.git
```

2. Open the directory:

```bash
    cd PC-Helper/
```

3. Run the [main.bat](https://github.com/IgorDGomes/PC-Basic-Needs/blob/main/main.bat) file.
